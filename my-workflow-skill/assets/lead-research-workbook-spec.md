# 리드 리서치 워크북 명세

## 시트

| 시트 | 목적 | 핵심 열 |
|---|---|---|
| 리드 리스트 | 마케팅 또는 영업이 넘긴 원본 | 리드 ID, 기업명, 본사 국가, 산업, 웹사이트, 유입 경로 |
| 리드 리서치 | 공개 근거와 제안 가설 | 공개 신호, URL, 확인일, 고객 정보 접점, 가설, 라이프사이클, CTA, 적합성, 우선순위 |
| 의사결정 큐 | 매니저의 실행 결정 | 최종 판단, 다음 액션, 담당, 목표일, 검토 메모 |
| 검토 가이드 | 점수·표현·검토 기준 | 기준, 기록 방식, 검토자 액션 |

## 입력 규칙

- 공개 신호와 제안 가설을 별도 열에 둔다.
- URL은 한 셀에 하나만 기록한다.
- 최종 판단: `미검토`, `승인`, `보류`, `제외`, `수정 요청`.
- 다음 액션: `리서치 보강`, `담당자 확인`, `소개 요청`, `데모 제안`, `보류`.
- 고객·개인 정보 열을 만들지 않는다.
